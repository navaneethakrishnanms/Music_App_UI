## ğŸŒŸ Customization

### Change Color Scheme
Edit CSS variables in `style.css`:

```css
:root {
    --bg-black: #000000;
    --sidebar-black: #000000;
    --main-bg: #121212;
    --card-hover: #282828;
    --spotify-green: #1db954;
    --text-gray: #b3b3b3;
}
```

### Add Your Own Playlists
Modify the playlist section in `index.html`:

```html
<div class="playlist-list">
    <p>Your Playlist Name</p>
</div>
```

### Change Images
Replace Unsplash URLs with your own image links in `index.html`
